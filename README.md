# Holiday Expenses Calculator

You can add expenses and calculate the payouts.

### 1. Install all packages
```
npm run install-all
```

### 2. Run the project
```
npm run start
```

### 3. Test backend
```
npm test
```

### 4. E2E Test
```
npm run e2e
```

or

```
npm run e2e-headless
```

### 5. Lint and Prettier
```
npm run prettier:check
```

and

```
npm run prettier:write
```
